## Validation Plan

The focused local browser-render validation for this evidence packet is:

```bash
npm run build && npx playwright test tests/inc001-render.spec.js
```

The full local owning-repo validation is:

```bash
npm run verify
```

`npm run verify` runs the browser suite locally in addition to the repository's
non-browser validation. The GitHub `Build & Test` workflow currently covers the
Python stats tests, arc JavaScript syntax check, arc unit tests, and JSDOM DOM
smoke test; it intentionally does not run Playwright browser tests because those
require a Chromium download. Therefore the local browser command above is the
render evidence for this packet, while GitHub CI remains the remote non-browser
repository gate.

The packet should be cited by `civilization-operation` only after the PR that
adds it has passed focused local browser validation, full local validation,
GitHub CI, exact-head adversarial review, and has been merged to `origin/main`.
